Bookmarklets

A compilation of bookmarklets. To use, drag the link to your bookmarks bar.

Change All Font

This bookmarklet changes all the font on a page by setting a style attribute on every element. It's useful for reading articles that use too thin of a font.

When choosing a new font, be careful to type in the name of the font correctly.

Safe Browsing

This bookmarklet creates a popup you can use to test the safety of urls. Testing a url will open up Google's transparency report of that page.

Text Analytics

After selecting a section of text, running this bookmarklet will return info on how many words you selected and how many characters are in that word. Also included are the average length of a word in the selection and the length of the smallest and largest words. Character count includes punctuation and apostraphes, but word count does not.
